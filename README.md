## 🗒 About

This is a Reminder App in which the tasks you give are completed according to your time.

## ⚙️ Features

- Main features are as follows:

    - Dark & Light Theme
    - Add Your Tasks & Time
    - Edit & Delete Tasks
    - SQLite DataBase
    - Notification Feature
    - MVC directory structure
